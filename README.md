# Standard Go Project Layout

`package` 是 Go 语言访问控制的最小粒度. Go 语言项目的最佳实践是按照职责对模块进行垂直拆分, 将代码按所属领域的不同分到多个 `package` 中.   

## 扩展目录
* `dist`: 存放本地编译的可执行文件(此文件夹不提交到代码仓库).
